Timestamp Microservice

This is a Timestamp microservice which can be called with a natural date or unix date timestamp as a parameter and returns a JSON with both the unix time stamp and natural language form of the given date.

Example usage:

    /March 11, 1952
    /-562035600

Example response:

    {"unix": -562035600,
     "natural": "March 11, 1952"}
